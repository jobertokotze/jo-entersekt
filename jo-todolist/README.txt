# Jo Todo list exercise

Here are 2 methods of running my app.

##################################################################################

1. Docker...

### Install

- Install https://docs.docker.com/engine/installation/

### Run
`docker login`
`docker run -p 4000:8080 jobertokotze/jo-todolist`

Visit http://localhost:4000 in your browser

##################################################################################

2. Run locally...

### Install

- Install https://nodejs.org/en/
- Navigate to my archive
- Unzip file and cd into it
- run `npm install`

### Run
`node app.js`

Visit http://localhost:8080 in your browser
